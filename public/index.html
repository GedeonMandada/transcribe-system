Admin Actions

Use these actions to manage the sermon index in the Redis database. These are powerful tools and should be used with care.

syncRe-index All Sermons

Scans R2 storage and rebuilds the search index. Use this if sermons are missing from the library.

delete_sweepClear Entire Index

Deletes all sermon metadata from the index. This cannot be undone without a re-index.
